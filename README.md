# 🌦️ Weather App

A modern and responsive weather application that provides real-time weather updates for any city in the world. Built using **HTML, CSS, and JavaScript**, this app integrates live data from the OpenWeather API and displays it with a clean UI and interactive map.

---

## 🚀 Live Demo

👉 **Try it here:** [https://hawadrip.netlify.app/](https://hawadrip.netlify.app/)

---

## ✨ Features

* 🌍 Search weather by city name
* 📍 Get weather using your current location
* 🌡️ Real-time temperature, humidity, wind speed, and feels-like data
* 🗺️ Interactive map using Leaflet.js
* 🌓 Light/Dark mode toggle
* 🔁 Recently searched cities (stored in localStorage)
* ⚡ Fast and responsive UI

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **API:** OpenWeather API
* **Maps:** Leaflet.js
* **Deployment:** Netlify

## 🔑 API Setup

This project uses **OpenWeather API**.

* Get your API key from: [https://openweathermap.org/api](https://openweathermap.org/api)
* Replace the API key in `script.js`:

```js
const API_KEY = "your_api_key_here";
```
